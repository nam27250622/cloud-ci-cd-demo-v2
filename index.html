Lý Thuyết CI/CD

CI/CD là gì?

CI/CD (Continuous Integration / Continuous Deployment) là một phương pháp phát triển phần mềm hiện đại giúp tự động hóa quy trình tích hợp code và triển khai ứng dụng.

  • CI (Continuous Integration): Tự động tích hợp code từ nhiều lập trình viên, chạy test để phát hiện lỗi sớm.
  • CD (Continuous Deployment/Delivery): Tự động triển khai code lên môi trường production sau khi build thành công.

GitHub Actions

GitHub Actions là công cụ CI/CD mạnh mẽ được tích hợp sẵn trong GitHub. Cho phép tạo các workflow (.yml) để tự động chạy các lệnh khi có sự kiện như push code, pull request...

Trong đề này, chúng ta sử dụng GitHub Actions để tự động build và deploy website lên GitHub Pages mỗi khi push code.

GitHub Pages

GitHub Pages là dịch vụ miễn phí của GitHub giúp host website tĩnh (HTML, CSS, JS). Khi kết hợp với GitHub Actions, website sẽ được tự động cập nhật mỗi khi có code mới được push lên repository.
