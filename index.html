Lý Thuyết CI/CD

CI/CD là gì?

CI/CD (Continuous Integration / Continuous Deployment) là một phương pháp phát triển phần mềm hiện đại giúp tự động hóa quy trình tích hợp code và triển khai ứng dụng.

  • CI (Continuous Integration): Tự động tích hợp code từ nhiều lập trình viên, chạy test để phát hiện lỗi sớm.
  • CD (Continuous Deployment/Delivery): Tự động triển khai code lên môi trường production sau khi build thành công.

GitHub Actions

GitHub Actions là công cụ CI/CD mạnh mẽ được tích hợp sẵn trong GitHub. Cho phép tạo các workflow (.yml) để tự động chạy các lệnh khi có sự kiện như push code.

GitHub Pages

GitHub Pages là dịch vụ miễn phí giúp host website tĩnh. Khi kết hợp với GitHub Actions, website sẽ được tự động deploy mỗi khi push code.

Các Bước Thực Hành - Đề 4

1
Tạo GitHub Repository và clone về máy.
2
Xây dựng website tĩnh bằng HTML, CSS, JavaScript.
3
Tạo file workflow .github/workflows/deploy.yml
4
Push code lên GitHub → Workflow tự động chạy
5
Enable GitHub Pages và kiểm tra website được deploy tự động
